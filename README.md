# Svelte Workspace

Веб-приложение для управления задачами. Проект задуман как многофункциональное рабочее пространство, которое со временем будет расширяться разным очень полезным (и не очень) функционалом.

**Статус**: 🌱 Прототип (нет адаптива, мало a11y, местами спорный UX)  
**Демо**: [svelte-workspace.vercel.app](https://svelte-workspace.vercel.app/)

## ✨ Основные возможности

- Отображение задач выбранного проекта;
- Добавление задачи;
- Изменение статуса задачи;
- ...

## 🚀 Технологический стек

- **Фронтенд:** SvelteKit (Svelte 5, Vite, TypeScript)
- **Бэкенд / База данных:** Firebase (?)
- **Стилизация:** PostCSS
- **Библиотеки:** DOMPurify
